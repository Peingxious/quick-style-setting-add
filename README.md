# Quick Style Settings Manager

This Obsidian plugin allows you to quickly manage (CRUD) CSS snippets designed for the [Style Settings](https://github.com/mgmeyers/obsidian-style-settings) plugin.

## Features

- **List Snippets**: View all your CSS snippets.
- **Create Snippet**: Create new CSS snippets with a pre-filled Style Settings template.
- **Edit Snippet**: Edit existing snippets directly within Obsidian.
- **Delete Snippet**: Remove unwanted snippets.
- **Template Support**: Easily insert the Style Settings configuration block.

## Usage

1. Open the Command Palette (`Ctrl/Cmd + P`).
2. Search for "Manage Style Settings Snippets".
3. Use the interface to manage your snippets.
